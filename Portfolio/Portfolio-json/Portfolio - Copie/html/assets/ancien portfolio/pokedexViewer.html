PokeDex Viewer

https://github.com/johanvadenne/pokedex_viewer

Description

Bienvenue sur le PokéDex Viewer, une application web simple qui vous permet d'explorer les 151 premiers Pokémon de la première génération. Cette application a été créée en utilisant une API Node.js.

Installation

npm install fs

npm install express

npm install cors

node .\server_pokedex.js

Ouvrez votre navigateur et accédez à http://127.0.0.1:5001 pour l'API explorer le PokéDex Viewer.

Ouvrez votre index.html avec votre navigateur pour explorer le PokéDex Viewer.
