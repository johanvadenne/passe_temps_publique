PYTHON

Python est un langage de programmation qui peut s'utiliser dans de nombreux
contextes et s'adapter à tout type d'utilisation grâce à des bibliothèques
spécialisées. Il est cependant particulièrement utilisé comme langage de script
pour automatiser des tâches simples mais fastidieuses, comme un script qui
récupérerait la météo sur Internet ou qui s'intégrerait dans un logiciel de
conception assistée par ordinateur afin d'automatiser certains enchaînements
d'actions répétitives. Son extension de fichier fini toujours par .py

Comment comprendre le langage python?

print("bonjour tout le monde, comment vous allez?")

"print" transmet le texte, nombre ou autre variable que vous avez demandé à dire.
il faut faire attention de mettre les guillemets ("") ou ('') lorsque l'on met
du texte, cependant ce n'est pas la peine lorsque l'on appelle des variable
Il n'y a pas besoin de mettre de point-virgule (;) à chaque fin de ligne.

CODE

texte1 = "bonjour j'ai "
nombre = 17
texte2 = " ans."
print(texte1, nombre, texte2)

bonjour j'ai 17 ans.

Ici je crée plusieurs variable que je vais ensuite réutiliser dans le script,
puis je les rassemble dans l'ordre pour construire mon texte.
Il faut faire attention de respecter les espaces, sinon le script collera les
chaines toutes les variables entre elles.
Voici un exemple:

CODE

texte1 = "bonjour j'ai"
nombre = 17
texte2 = "ans."
print(texte2, nombre, texte2)

bonjour j'ai17ans.

variables

  • bool : booléen (True ou False) si x = 9 < 10 donc x = true
  • int : entier. x = 12, x= 45, x = 56984
  • float : nombre flottant qui a la précision d'un double. x = 5.5, x = 7.654
  • str : chaîne de caractère (string). x = "Bonjour tout le monde!"
