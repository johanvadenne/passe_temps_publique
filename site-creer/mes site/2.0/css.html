<link rel="stylesheet" type="text/css" href="style.css">: permet de relié un fichier css à un fichier html
color: permet de changer de couleur le texte (couleur, rgb, rgba, hex, hls, hlsa)
text-aligne: permet de positionner le texte, center centre le texte
class="": utilisé pour pointer vers un nom de classe (créer un groupe), .green
id="": spécifie un identifiant unique, #violet
*: selection tous les éléments
