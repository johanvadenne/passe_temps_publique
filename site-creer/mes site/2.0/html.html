<form>: peut être affiché de plusieurs façons, selon l' type
type="texte": définit un champ de saisie
type="checkbox": définit un botton à choix multiples
type="texte": définit un boutton à choix unique
<select>: crée une liste déroulente
<option>: élément de la liste déroulente
size="": nombre d'élément de la liste visible
multiple: choix multiple ctr + clic gauche
<textarea>: champ de saisie multiligne
rows="": nombre de ligne visible
cols="": nombre de collone visible
type="submit": soumet le formulaire

<form action="">: attribut définit l'action à effectuer lorsque le formulaire est soumis
<form action="" method="">: attribut spécifie la méthode HTTP à utiliser lors de la soumission des données du formulaire
<form action="" autocomplete="">: attribut spécifie si un formulaire doit avoir la saisie semi-automatique activée ou désactivée
<form action="" novalidate>: attribut est un attribut booléen

<input type="button">: définit un bouton
<input type="checkbox">: définit une case à cocherv
<input type="color">: est utilisé pour les champs de saisie qui doivent contenir une couleur
<input type="date">: est utilisé pour les champs de saisie qui doivent contenir une date
<input type="datetime-local">: spécifie un champ de saisie de date et d'heure, sans fuseau horaire
<input type="email">: est utilisé pour les champs de saisie qui doivent contenir une adresse e-mail
<input type="file">: définit un champ de sélection de fichier et un bouton "Parcourir" pour les téléchargements de fichiers
<input type="hidden">: définit un champ de saisie masqué (invisible pour un utilisateur)
<input type="image">: définit une image comme un bouton d'envoi
<input type="month">: permet à l'utilisateur de sélectionner un mois et une année
<input type="number">: définit un champ de saisie numérique
<input type="password">: définit un champ mot de passe
<input type="radio">: définit un bouton radio
<input type="range">: définit un contrôle permettant de saisir un nombre dont la valeur exacte n'a pas d'importance (comme un curseur)
<input type="reset">: définit un bouton de réinitialisation qui réinitialisera toutes les valeurs du formulaire à leurs valeurs par défaut
<input type="search">: est utilisé pour les champs de recherche
<input type="submit">: définit un bouton pour soumettre des données de formulaire à un gestionnaire de formulaire
<input type="tel">: est utilisé pour les champs de saisie qui doivent contenir un numéro de téléphone
<input type="text">: définit un champ de saisie de texte sur une seule ligne
<input type="time">: permet à l'utilisateur de sélectionner une heure (pas de fuseau horaire)
<input type="url">: est utilisé pour les champs de saisie qui doivent contenir une adresse URL
<input type="week">: permet à l'utilisateur de sélectionner une semaine et une année
