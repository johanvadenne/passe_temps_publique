PHP

PHP (officiellement, ce sigle est un acronyme récursif pour PHP Hypertext
Preprocessor) est un langage de scripts généraliste et Open Source, spécialement
conçu pour le développement d'applications web. Il peut être intégré facilement au HTML.
son extension fini toujours par .php

Comment comprendre le langage php?

CODE

<?php
// je me présente
$name = "Devro ";
$prenom = "Tom ";
$age = 17;
echo "\nMon nom est ". $name;
echo "\net mon prénom ". $prenom;
echo "\nj'ai actuellement ". $age." ans.";

commençons par la première ligne "<?php", cela est essentielle pour commencer
un programme en php. Si vous oublié cela votre code ne marchera pas. Pour créer une variable il faut commencer par un "$" puis le nom de la variable.
Pour afficher un texte on utilise "echo" puis j'écris ce que je veux après.

variables

  • bool : booléen (True ou False) si $x = 9 < 10 donc $x = true
  • int : entier. $x = 12, $x = 45, $x = 56984
  • float : nombre flottant qui a la précision d'un double. $x = 5.5, $x = 7.654
  • str : chaîne de caractère (string). $x = "Bonjour tout le monde!"
